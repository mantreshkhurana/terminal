# Terminal

This is a simple yet customizable terminal emulator written in Flutter.

## Screenshots

![Screenshot 1](./screenshots/screenshot-1.png)

## Supported OSs

- [x] macOS
- [x] Linux
- [x] Windows

## Features

- [ ] oh-my-zsh support
- [ ] Terminal plugins
- [ ] Customizable
- [ ] Customizable key bindings
- [ ] Customizable themes
- [ ] Customizable fonts
- [ ] Customizable cursor
- [ ] Customizable shell
- [ ] Customizable shell arguments
- [ ] Customizable shell environment variables
- [ ] Customizable shell aliases

## Installation

```bash
git clone https://github.com/spyxpo/terminal.git
cd terminal
flutter pub get
```

## Usage

```bash
flutter run
```
